# URL Shortener

### Technologies
* Slim Framework
* Phinx
* Twig
* HTML 5
* Bulma CSS
* JS
* PHPUnit
* PHP-DI

### Requirements
* PHP 8.4
* PHP Extensions (pdo, sqlite)
* Composer

### Get Started
```bash 
  composer install
```
```bash 
  composer start-dev
```
### Tests
```bash
  composer test
```
